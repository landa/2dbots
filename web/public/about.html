2dBots — Robot Coding Game

2dBots is a series of two-dimensional robot challenges that teach you algorithms and programming.


2dBots teaches you how to code by walking you through a series of tutorial and challenges. By the time you're done, you'll have written the code that can control a pretty sophisticated two-dimensional bot!

Here's why 2dBots is so awesome:

  • 2dBots uses a very popular (and real) language called JavaScript
  • 2dBots teaches you important programming concepts...
  • ...but it also teaches you important algorithms from the field of computer science
  • 2dBots is really fun, seeing as it's actually a game

By the time you're done with 2dBots, you'll be able to build the algorithm for a robot like the one you see on the right.



What if I'm already a really good coder?


No problem. Some of the challenges are really hard, so give it your best try!
